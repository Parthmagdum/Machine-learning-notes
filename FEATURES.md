## **OOP Concepts**

|--------------|---------------------------------|----------------------------------|

## **Class & Object**

- **Class** = blueprint  
- **Object** = real instance of class  

---

## **Constructor**

Special method to initialize object (same name as class, no return type)

| Type | Meaning |
|------|--------|
| Default | No parameters |
| Parameterized | Has parameters |
| Constructor Overloading | Multiple constructors with different params |

---

## **this vs super**

| this | super |
|------|------|
| Refers to current class | Refers to parent class |
| Access current variables/methods | Access parent variables/methods |
| Call current constructor | Call parent constructor |

---

## **Overloading vs Overriding**

| Overloading | Overriding |
|-------------|-----------|
| Same method name, different parameters | Same method name & parameters |
| Compile-time polymorphism | Runtime polymorphism |
| Same class | Inheritance required |
