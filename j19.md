### Q. Define an array. Explain types of array with example. (8 Marks)

An **array** in Java is a **collection of elements of the same data type** stored in **contiguous memory locations**.  
Arrays allow storing multiple values under a single variable name and are accessed using index (starting from 0).

---

### ✅ Types of Arrays in Java

---

#### 1. **One-Dimensional Array**
Stores data in a single row (linear form).

**Example:**
```java
class Demo {
    public static void main(String[] args) {
        int arr[] = {10, 20, 30};

        for(int i = 0; i < arr.length; i++) {
            System.out.println(arr[i]);
        }
    }
}

Output:

10
20
30


---

2. Two-Dimensional Array (Matrix)

Stores data in rows and columns (table form).

Example:

class Demo {
    public static void main(String[] args) {
        int arr[][] = {
            {1, 2},
            {3, 4}
        };

        for(int i = 0; i < 2; i++) {
            for(int j = 0; j < 2; j++) {
                System.out.print(arr[i][j] + " ");
            }
            System.out.println();
        }
    }
}

Output:

1 2
3 4


---

3. Jagged Array

Array of arrays with unequal column size.

Example:

class Demo {
    public static void main(String[] args) {
        int arr[][] = {
            {1, 2},
            {3, 4, 5}
        };

        for(int i = 0; i < arr.length; i++) {
            for(int j = 0; j < arr[i].length; j++) {
                System.out.print(arr[i][j] + " ");
            }
            System.out.println();
        }
    }
}

Output:

1 2
3 4 5


---

Arrays provide fast access, efficient memory usage, and organized data storage for multiple values.
