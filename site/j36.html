Q. Write short note on throws keyword. (4 Marks)

The throws keyword in Java is used in method declarations to indicate that the method may throw one or more exceptions.
It informs the caller of the method to handle the exception, instead of handling it inside the method itself.

It is mainly used for checked exceptions such as IOException, SQLException, etc.

✅ Example

```java import java.io.*;

class Demo { static void readFile() throws IOException { // This method may throw IOException throw new IOException("File error"); }

public static void main(String[] args) {
    try {
        readFile();
    } catch (IOException e) {
        System.out.println("Exception handled: " + e.getMessage());
    }
}

}

✅ Output

Exception handled: File error

The throws keyword helps propagate exceptions and organize error handling in Java programs.
