Q. What is constructor? Explain types of constructor with example. (8 Marks)

A constructor in Java is a special method used to initialize objects.
It has the same name as the class, no return type, and is automatically called when an object is created.

✅ Types of Constructors in Java

1. Default Constructor

  • Created by compiler if no constructor is defined.
  • Initializes object with default values.

Example: ```java class Demo { int a;

// Default constructor
Demo() {
    a = 10;
    System.out.println("Default Constructor");
}

public static void main(String[] args) {
    Demo d = new Demo();  // calls default constructor
    System.out.println("a = " + d.a);
}

}

Output:

Default Constructor a = 10

  1. Parameterized Constructor

Takes parameters to initialize objects with custom values.

Example:

class Student { String name; int age;

// Parameterized constructor
Student(String n, int a) {
    name = n;
    age = a;
}

public static void main(String[] args) {
    Student s = new Student("Amit", 19);
    System.out.println(s.name + " " + s.age);
}

}

Output:

Amit 19

Constructors help in initializing objects, improving code clarity and control over object creation.
