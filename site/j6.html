Q. Describe break and continue statements in Java. (4 Marks)

break and continue are control flow statements used in loops and switch statements to control program execution.

break Statement

  • Stops the loop or switch immediately.
  • Control jumps to the next statement after the loop/switch.

Example:

for (int i = 1; i <= 5; i++) {
    if (i == 3) break;
    System.out.println(i);
}
// Output: 1 2

continue Statement

Skips the current iteration of the loop.

Control jumps to the next iteration without stopping the loop.

Example:

for (int i = 1; i <= 5; i++) {
    if (i == 3) continue;
    System.out.println(i);
}
// Output: 1 2 4 5
