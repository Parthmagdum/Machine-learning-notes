Difference Between Permutation and Combination – 4 Marks

Basis Permutation Combination
Definition Arrangement of objects in a specific order. Selection of objects without considering order.
Order Importance Order matters (AB ≠ BA). Order doesn’t matter (AB = BA).
Formula ( nP_r = \dfrac{n!}{(n - r)!} ) ( nC_r = \dfrac{n!}{r!(n - r)!} )
Meaning Number of possible arrangements of items. Number of possible selections of items.
Example For 3 letters A, B, C — 2-letter permutations are AB, BA, AC, CA, BC, CB (6 ways). For the same letters, 2-letter combinations are AB, AC, BC (3 ways).
Use Case Used when sequence or order is important (e.g., passwords, rankings). Used when selection is important (e.g., choosing a team, lottery).

✅ In short:
- Permutation = Arrangement (Order matters)
- Combination = Selection (Order doesn’t matter)
