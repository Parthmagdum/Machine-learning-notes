Q. Write short note on Stack class. (4 Marks)

The Stack class in Java is a LIFO (Last In, First Out) data structure, meaning the last element inserted is the first one removed. It is part of the java.util package and is used to store and manage data in a stack format.

Key Methods of Stack: - push() → Adds an element to the stack - pop() → Removes and returns the top element - peek() → Returns the top element without removing it - empty() → Checks if the stack is empty - search() → Searches an element and returns its position

Example: ```java import java.util.Stack;

class Demo { public static void main(String[] args) { Stack s = new Stack<>(); s.push(10); s.push(20); System.out.println(s.pop()); // Output: 20 } }

The Stack class is useful for expression evaluation, backtracking, recursion, and undo operations.
