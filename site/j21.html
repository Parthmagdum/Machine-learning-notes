Q. Explain with example ArrayList and LinkedList class in Java. (8 Marks)

Both ArrayList and LinkedList are classes in Java used to store dynamic data. They are part of the java.util package and implement the List interface.

✅ ArrayList

  • Based on dynamic array
  • Allows fast access (indexing)
  • Slower in inserting/deleting in the middle
  • Suitable when read/lookup operations are frequent

Example: ```java import java.util.ArrayList;

class Demo { public static void main(String[] args) { ArrayList list = new ArrayList<>(); list.add("A"); list.add("B"); list.add("C");

    System.out.println(list);  // Output: [A, B, C]
}

}

✅ LinkedList

Based on doubly linked list

Faster for insertion/deletion

Slower in accessing elements by index

Suitable when modifying (add/remove) elements frequently

Example:

import java.util.LinkedList;

class Demo { public static void main(String[] args) { LinkedList list = new LinkedList<>(); list.add("X"); list.add("Y"); list.add("Z");

    System.out.println(list);  // Output: [X, Y, Z]
}

}

✅ Difference (Short Summary)

Feature ArrayList LinkedList

Storage Dynamic array Doubly linked list Access Speed Fast Slow Insert/Delete Slow Fast Use Case More read operations Frequent insert/delete

These classes provide dynamic storage and flexible data handling during runtime.
