Q. Explain multidimensional array with example. (4 Marks)

A multidimensional array in Java is an array of arrays. It is used to store data in a matrix (row–column) form or more than one dimension. The most common form is a two-dimensional array.

✅ Example: 2D Array (Matrix)

```java class Demo { public static void main(String[] args) { int arr[][] = { {1, 2, 3}, {4, 5, 6} };

    for(int i = 0; i < arr.length; i++) {
        for(int j = 0; j < arr[i].length; j++) {
            System.out.print(arr[i][j] + " ");
        }
        System.out.println();
    }
}

}

Output:

1 2 3 4 5 6

Multidimensional arrays allow storing data in rows and columns, making them suitable for tables, matrices, and grid-based data.
