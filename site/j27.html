Q. Difference between Method Overloading and Method Overriding. (4 Marks)

Method Overloading Method Overriding
Same method name, different parameters Same method name, same parameters
Occurs within the same class Occurs between parent and child class (inheritance)
Known as compile-time polymorphism Known as runtime polymorphism
Return type may be same or different Return type must be same
Parameters must be different (type/number/order) Parameters must be identical
@Override annotation not used @Override annotation often used
Used to increase readability & flexibility Used to modify/inherit parent method behavior

✅ Example (Simple)

Overloading ```java void show(int a) {} void show(String b) {}

Overriding

class A { void show() {} } class B extends A { void show() {} }
