Q. Explain in brief multiple catch block with example. (4 Marks)

In Java, a multiple catch block is used when a single try block can throw different types of exceptions.
Multiple catch blocks allow handling each exception type separately based on the error.

This improves error handling and prevents program termination due to unexpected exceptions.

✅ Example

```java class Demo { public static void main(String[] args) { try { int a = 10 / 0; // ArithmeticException int arr[] = new int[3]; arr[5] = 20; // ArrayIndexOutOfBoundsException } catch (ArithmeticException e) { System.out.println("Arithmetic Exception Occurred"); } catch (ArrayIndexOutOfBoundsException e) { System.out.println("Array Index Out of Bounds"); } catch (Exception e) { System.out.println("General Exception"); } } }

✅ Output

Arithmetic Exception Occurred

Multiple catch blocks allow specific handling of different exceptions, improving program stability.
