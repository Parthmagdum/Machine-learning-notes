Q. Define a constructor. Explain constructor overloading in Java with example. (8 Marks)

A constructor in Java is a special method used to initialize objects.
It has the same name as the class, no return type, and is executed automatically when an object is created.

✅ Constructor Overloading

Constructor overloading means having multiple constructors in the same class with different parameter lists.
It allows objects to be created in multiple ways.

✅ Example: Constructor Overloading

```java class Person { String name; int age;

// Default constructor
Person() {
    name = "Unknown";
    age = 0;
}

// Parameterized constructor (1 parameter)
Person(String n) {
    name = n;
    age = 0;
}

// Parameterized constructor (2 parameters)
Person(String n, int a) {
    name = n;
    age = a;
}

void display() {
    System.out.println(name + " " + age);
}

public static void main(String[] args) {
    Person p1 = new Person();
    Person p2 = new Person("Rahul");
    Person p3 = new Person("Amit", 21);

    p1.display();
    p2.display();
    p3.display();
}

}

✅ Output

Unknown 0 Rahul 0 Amit 21

Constructor overloading provides flexibility in creating objects with different initial values.
