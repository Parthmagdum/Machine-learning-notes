Q. Define interface. Explain how to define and implement interface with example. (8 Marks)

An interface in Java is a reference type that contains abstract methods, constants, and default/static methods.
It is used to achieve 100% abstraction (before Java 8) and multiple inheritance in Java.
A class implements an interface to provide the method definitions.

✅ How to Define and Implement Interface

Step 1: Define an Interface

Use the interface keyword.

Step 2: Implement Interface in a Class

Use the implements keyword.

✅ Example

```java // Defining interface interface Animal { void sound(); // abstract method }

// Implementing interface class Dog implements Animal { public void sound() { System.out.println("Dog barks"); } }

public class Main { public static void main(String[] args) { Dog d = new Dog(); d.sound(); } }

✅ Output

Dog barks

Interfaces help in achieving abstraction, multiple inheritance, and standardizing method structure across classes.
